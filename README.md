# Análisis de datos de ventas – Alura Store

## Descripción del proyecto

Este proyecto consiste en un análisis exploratorio de datos de ventas de cuatro tiendas pertenecientes a Alura Store.
El objetivo del análisis es evaluar el rendimiento de cada tienda utilizando diferentes indicadores para determinar cuál presenta el menor desempeño y recomendar cuál debería vender el Sr. Juan para iniciar un nuevo emprendimiento.

## Objetivos del análisis

* Analizar la facturación total de cada tienda
* Identificar las categorías de productos más vendidas
* Evaluar la calificación promedio otorgada por los clientes
* Analizar los productos más y menos vendidos
* Comparar el costo promedio de envío entre las tiendas

## Herramientas utilizadas

* Python
* Pandas
* Matplotlib
* Google Colab
* GitHub

## Análisis realizados

Durante el análisis se estudiaron diferentes métricas de desempeño de las tiendas, incluyendo:

* Facturación total por tienda
* Ventas por categoría de producto
* Calificación promedio de clientes
* Productos más y menos vendidos
* Costo promedio de envío

Los resultados se visualizaron mediante gráficos para facilitar la comparación entre tiendas.

## Conclusión

Luego de analizar los datos disponibles, se observa que la **Tienda 4 presenta la menor facturación total** en comparación con las demás tiendas.
Aunque las calificaciones de los clientes y los costos de envío son similares entre las tiendas, el menor nivel de ingresos indica que su rendimiento general es inferior.

En base a estos resultados, se recomienda considerar la **venta de la Tienda 4**, permitiendo concentrar los recursos en las tiendas con mejor desempeño.
